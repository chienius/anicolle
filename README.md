## 安装与使用

很简单，clone下这个版本库，运行 `pip install -r requirements.txt`

然后 `run.py -h` 就好喇！

目前支持 漫游BT服务器 的更新检查，添加番剧时将检索关键词一栏写上要在 漫游 搜索的番剧名字即可。

另外，目前的版本可能不支持 v0.1 的数据库，请删除数据库重来一次！
