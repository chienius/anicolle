# 安装与使用

很简单，clone下这个版本库，运行main.py即可。

由于程序提供了 Web UI ，可能需要额外安装提供了 Web UI 中间件的 Bottle 框架。

然后 `main.py -h` 就好喇！

目前支持 漫游BT服务器 的更新检查，添加番剧时将检索关键词一栏写上要在 漫游 搜索的番剧名字即可。
